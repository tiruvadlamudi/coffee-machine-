 Coffee Machine Simulation (Python)

A console-based Coffee Machine application developed using Python.
This project simulates a real coffee machine where users can order drinks,
insert coins, and receive coffee based on available resources.

 Technologies Used
- Python 3

 Features
- Menu-driven coffee selection
- Resource availability check
- Coin-based payment system
- Change calculation
- Admin report option
- Profit tracking

How to Run the Project
1. Install Python 3
2. Clone the repository
3. Run the program:

```bash
python coffee_machine.py
